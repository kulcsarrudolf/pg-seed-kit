# pg-seed-kit

A lightweight, zero-dependency seeder toolkit for Postgres that works with your ORM of choice: Prisma, Drizzle, TypeORM, or Sequelize.

Run one-time seed scripts on app startup, track execution status, and manage seeders via a small CLI.
The library ships **no runtime dependencies**: it borrows the database connection your ORM already owns through a tiny adapter.

> Under active development. Full documentation and the ORM-by-ORM guide are on the way.
